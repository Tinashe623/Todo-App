The project follows a **Modular Component Architecture**, ensuring high maintainability and scalability:

- **`TodoList.jsx` (Container)**: Orchestrates the application state and business logic.

- **`TaskItem.jsx`**: Manages the individual task lifecycle, including editing and completion states.
